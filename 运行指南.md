# 工业级低代码计算机视觉系统 - 平台启动指南

本指南将帮助您在以后轻松启动和访问本平台。

## 系统架构

本平台分为两部分：
1. **后端服务**：基于 FastAPI 构建，提供 API 接口和计算机视觉功能
2. **前端应用**：基于 Vue.js 和 Quasar 框架构建，提供用户界面

## 启动步骤

### 1. 启动后端服务

#### 方法一：使用启动脚本（推荐）

**Windows 系统：**
1. 双击运行 `start.bat` 文件
2. 脚本会自动：
   - 检查并创建虚拟环境（如果不存在）
   - 激活虚拟环境
   - 安装依赖（如果未安装）
   - 启动后端服务
python -m uvicorn src.backend.main:app --host 0.0.0.0 --port 8000 --reload
**PowerShell 或 Linux/Mac 系统：**
1. 在终端中执行 `start.ps1` 脚本（PowerShell）或 `start.sh` 脚本（Linux/Mac）
2. 脚本会自动完成与上述相同的操作

#### 方法二：手动启动

1. 打开终端/命令提示符
2. 进入项目根目录
3. 激活虚拟环境：
   ```bash
   # Windows
   .venv\Scripts\activate.bat
   
   # Linux/Mac
   source .venv/bin/activate
   ```
4. 安装依赖（如果未安装）：
   ```bash
   pip install -r requirements.txt
   ```
5. 启动后端服务：
   ```bash
   python -m uvicorn src.backend.main:app --host 0.0.0.0 --port 8000 --reload
   ```

### 2. 启动前端服务

1. 打开另一个终端/命令提示符
2. 进入项目根目录
3. 确保 Node.js 和 npm 已正确安装
4. 如果使用的是 Anaconda 环境，确保添加到 PATH：
   ```bash
   # Windows PowerShell
   $env:PATH += ";E:\anaconda3\envs\yolo-detect"
   ```
5. 安装前端依赖（如果未安装）：
   ```bash
   npm install
   ```
6. 启动前端服务：
   ```bash
   npm run dev
   ```

### 3. 访问平台

1. 后端服务启动后，会在控制台显示访问地址，通常为：`http://127.0.0.1:8000`
2. 前端服务启动后，会在控制台显示访问地址，通常为：`http://localhost:3080/`
3. 打开浏览器，访问前端服务地址 `http://localhost:3080/` 即可使用平台

## 常见问题与解决方案

### 1. 虚拟环境问题

- **问题**：启动脚本无法创建或激活虚拟环境
- **解决方案**：手动创建虚拟环境并安装依赖

### 2. Node.js/npm 路径问题

- **问题**：无法找到 node 或 npm 命令
- **解决方案**：确保 Node.js 已正确安装并添加到系统 PATH，或使用完整路径执行命令

### 3. 端口占用问题

- **问题**：端口 8000（后端）或 3080（前端）已被占用
- **解决方案**：修改配置文件中的端口号，或停止占用端口的其他进程

### 4. 依赖安装问题

- **问题**：某些依赖无法安装
- **解决方案**：检查网络连接，或尝试使用国内镜像源安装

## 系统关闭

1. 关闭前端服务：在前端服务终端中按 `Ctrl + C`
2. 关闭后端服务：在后端服务终端中按 `Ctrl + C`
3. 如果使用了虚拟环境，执行 `deactivate` 命令退出虚拟环境
